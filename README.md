# Recipe App

A modern Recipe App built using **Kotlin Multiplatform (KMP)** and **Jetpack Compose** with a dummy API integration.

The app supports recipe management with search, filters, pagination, and detailed recipe views.

---

## Features
- Recipe Listing
- Add, Update & Delete Recipes
- Recipe Details Screen
- Search Recipes
- Filter Recipes by Tags
- Sort Recipes
- Pagination Support
- Cross-platform architecture using KMP
- Modern UI with Jetpack Compose
- Dependency Injection with Koin

---

## Architecture
UI (Compose) → ViewModel → Repository → Dummy API

---

## Tech Stack
- Kotlin Multiplatform (KMP)
- Jetpack Compose
- MVVM Architecture
- Coroutines & Flow
- Dummy REST API
- Koin DI

---

## Main Components
- Recipe Listing Screen
- Recipe Details Screen
- Search & Filter Module
- Pagination Handling
- Repository Layer
- Shared KMP Module

---

## Functionalities
- Search recipes by name
- Filter recipes using tags
- Sort recipes dynamically
- Load recipes with pagination
- Perform CRUD operations
